我平常喜欢在微信公众号看些文章, 但是这时候如果有人发消息过来, 手机震了一下.......但是你并不知道是谁发来的消息, 这时候,按照微信培养的用户习惯....置顶保存文章,然后点击n个返回按钮然后点击`close`返回消息页面.....回了消息然后在回去看文章.....

so,这就是痛点所在,不能快速查看回复消息.

搞起来....

### 功能设想

在任意页面, 当接收到异步消息, 通知当前页面弹出一个按钮提示, 点击按钮 push 对应聊天页面, pop 可返回原来的页面.

*PS:按钮右滑可以消除*
